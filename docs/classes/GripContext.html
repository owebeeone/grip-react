    Class GripContext

    GripContext represents a node in the hierarchical parameter graph.

    Each context can have multiple parents with priorities, enabling complex inheritance patterns. Contexts provide the scope for parameter resolution and serve as attachment points for Taps (data producers).

    The context hierarchy determines which Tap provides data for a given Grip through the "closest provider" resolution algorithm.

    Class MultiAtomValueTap<Outs>

    Multi-value atom Tap that manages multiple Grips in a single container.

    MultiAtomValueTap extends the single-value concept to handle multiple related Grips efficiently. It provides both individual and bulk operations for managing the values, with automatic change detection and notifications.

    The Tap supports flexible initialization where values can be provided through a Grips array, a value map, or a combination of both.

          gripsMap:
              | undefined
              | Map<Values<Outs>, undefined | GripValue<Values<Outs>>>,
          opts?: { handleGrip?: Grip<MultiAtomTapHandle<Outs>> },
      ): MultiAtomValueTap<Outs>

      Creates a new MultiAtomValueTap instance.

      Merge semantics for initialization:

      • Tracked set S = union(grips, gripsMap?.keys())
      • For g in S:
        • if gripsMap has g: use that value (even if undefined)
        • else if g is in grips: use g.defaultValue
        • else: leave undefined

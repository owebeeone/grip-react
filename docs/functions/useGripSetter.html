      Returns a stable setter function for an AtomTap handle.

      This hook provides a convenient way to get a setter function that can update values managed by an AtomTap. The setter is stable across re-renders and safely handles cases where the handle isn't available yet.

      Type Parameters

      • T

        The type of value managed by the AtomTap

      Parameters

      • tapGrip: Grip<AtomTapHandle<T>>

        The Grip that provides the AtomTap handle

      • Optionalctx: GripContext | GripContextLike

        Optional context for data resolution

      Returns (v: T) => void

      A setter function for the AtomTap handle

      Example

      const setLocation = useGripSetter(WEATHER_LOCATION_TAP);

      return (
        <select onChange={e => setLocation(e.target.value)}>
          <option value="nyc">New York</option>
          <option value="london">London</option>
        </select>
      );

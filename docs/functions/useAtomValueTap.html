      Creates and manages an atom-style Tap for reactive state management.

      This hook creates an AtomValueTap that provides a specific Grip and returns a controller interface for reading and writing values. The Tap is automatically registered with the GRIP system and will notify subscribers when values change.

      The hook provides overloads to handle nullable vs non-nullable types appropriately. When an initial value is provided and is non-nullable, the returned AtomTap will have a non-nullable type.
